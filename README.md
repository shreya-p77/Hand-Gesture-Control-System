# Hand Gesture Controlled Browser Automation System

## Project Overview

This project uses Computer Vision and Hand Gesture Recognition to control browser actions using a webcam. The system detects hand gestures in real-time and performs predefined actions such as opening websites and closing browsers.

## Features

* Open Claude using 1 finger gesture
* Open ChatGPT using 2 finger gesture
* Open Gemini using 3 finger gesture
* Open LinkedIn using 4 finger gesture
* Close all browsers using a fist gesture
* Real-time hand tracking using MediaPipe
* Automatic hand model download

## Technologies Used

* Python
* OpenCV
* MediaPipe
* Computer Vision
* Browser Automation

## Installation

```bash
pip install opencv-python mediapipe
```

## Run the Project

```bash
python td_gesture.py
```
